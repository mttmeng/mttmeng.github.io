环境配置

安装Homebrew

$ ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
(# 常见包的安装请参考官方教程 https://brew.sh/)
$ brew install coreutils

安装Node.js 6

$ brew install node@6
$ ln -s /usr/local/Cellar/node@6/6.x.x/bin/node /usr/local/bin
$ npm install npm@4 gulp@3 -g

问题：
1、 在第一步，如果是第一次运行node@6，需要运行 export PATH=”/usr/local/opt/node@6/bin:$PATH” 添加一些必须文件。

2、在第三步不要忽略install之前的npm，否则安装失败，要重新安装。
若加载过多，则存在异常，检查一下npm版本是否正确，重新运行命令行。
以下是正确结果。

安装Tomcat@7.8.72

1、首先到官网下载Tomcat：https://tomcat.apache.org/download-90.cgi

2、解压tomcat文件,最好把它文件名重命名为”Tomcat”，方便以后查找，最后把它放入/Library(资源库中)
(1).点击finder–>用户–>你电脑的名字–>资源库(有的也叫/Library)。
(2).有些苹果将library目录隐藏起来了，可以直接点左上角前往，前往文件夹，路径填 /资源库，就进入了资源库了。

3、用终端直接打开Tomcat
(1).进入Tomcat的bin目录下：
输入cd /Library/Tomcat/bin ，回车。(也可以打开Tomcat文件夹，把bin文件夹直接拖拉到终端，但要记得先输入cd+空格)
(2).授权bin目录下的所有操作：
输入sudo chmod 755 *.sh，回车
(3).输入密码，回车
(4).开启Tomcat，输入sudo sh ./startup.sh，回车

4、到浏览器输入网址localhost，若出现了下面的画面就证明成功了

配置~/.m2/settings.xml

在终端输入命令行：open ~/.m2，将下载下来的settings.xml文件放入子文件夹中。

如何在Markdown中引用本地图片

除了github、博客这两种方式外，还有两外一种简单的方法：

  1. 管理自己的图片：在有道云笔记中新建一个目录/文件夹——我的图片；
  2. 上传图片到有道云笔记上的”我的图片”目录下；
  3. 分享图片，会得到1个文件的分享URL，复制这个URL，在浏览器中打开，会看到图片；
  4. 右击图片 - 复制图片地址。
  5. 在Markdown文件中引用这个图片地址。
