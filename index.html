react - 列表和key

1、列表

使用 JavaScript 的 map() 方法来创建列表。

但这样写Chrome控制台会提示我们，应该给列表中的每个元素添加一个名为key的属性。

2、key

react使用key属性来标记列表中的每个元素，当列表发生变化时，react可以通过key知道哪些元素发生了变化，从而只重新渲染发生变化的元素，提高渲染效率。

一个元素的 key 最好是这个元素在列表中拥有的一个独一无二的字符串。通常，我们使用来自数据的 id 作为元素的 key

当元素没有确定的 id 时，你可以使用他的序列号索引 index 作为 key。
但并不推荐使用索引作为key，因为一旦列表中的数据发生重排，数据的索引也会发生改变，不利于react的渲染。

弹框遮罩的两种实现方式

第一种方法：空div作为遮罩

优点：可以方便的用一个开关控制遮罩的有无。
缺点：react组件中不支持空标签，故用这种方法会报错。

第二种方法：将弹框放在遮罩div中

优点：在适用于其他框架的同时，也能够适用于react;
可以通过shadow的opacity来控制遮罩样式的有无（如果不想要遮罩样式，可将遮罩的opacity改为0）

常用快捷键

1、空格（space）键

在Mac系统里，几乎所有文件都可以直接用空格键快速查看预览。拿图片举例，只要点选文件夹里某张图片，点击空格键就打开了，再点击上下左右键可切换浏览其他图片。

2、Command ⌘

⌘ + C 复制
⌘ + V 粘贴
⌘ + X 剪切
⌘ + option + V 移动
⌘ + A 全选
⌘ + delete 删除 相当于Windows下的Ctrl + D
⌘ + S 保存
⌘ + shift + 4 截图
⌘ + W 关闭当前的软件窗口（软件并没有真正退出进程），相当于点了右上角的红色叉叉。
⌘ + Q 真正退出软件。
⌘ + option + esc 强制退出某个软件。通常在软件无响应时使用。

⌘ + Z 撤销
刚删除了一张图片，可以点开废纸篓，找到刚刚删除的图片，然后放回原处。或者是在删除后马上点击 ⌘+Z，这时刚删除的图片回到原来的位置！

⌘ + N 新建
新建正在用的东西。比如正在用 Safari，那 ⌘+N 就是新建一个 Safari 窗口。在用 Photoshop，⌘+N 就是新建一个 PS 文稿。
control + 空格键 切换输入法。（也可以自己设置）
⌘ + 空格键 显示或隐藏 Spotlight 搜索栏。
⌘ + tab 切换app。
有些快捷键可以到系统偏好设置里修改成你喜欢（习惯）的，比如输入法，Spotlight 等。

安装Homebrew

$ ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
(# 常见包的安装请参考官方教程 https://brew.sh/)
$ brew install coreutils

安装Node.js 6

$ brew install node@6
$ ln -s /usr/local/Cellar/node@6/6.x.x/bin/node /usr/local/bin
$ npm install npm@4 gulp@3 -g

问题：
1、 在第一步，如果是第一次运行node@6，需要运行 export PATH=”/usr/local/opt/node@6/bin:$PATH” 添加一些必须文件。

2、在第三步不要忽略install之前的npm，否则安装失败，要重新安装。
若加载过多，则存在异常，检查一下npm版本是否正确，重新运行命令行。

安装Tomcat@7.8.72

1、首先到官网下载Tomcat：https://tomcat.apache.org/download-90.cgi

2、解压tomcat文件,最好把它文件名重命名为”Tomcat”，方便以后查找，最后把它放入/Library(资源库中)
(1).点击finder–>用户–>你电脑的名字–>资源库(有的也叫/Library)。
(2).有些苹果将library目录隐藏起来了，可以直接点左上角前往，前往文件夹，路径填 /资源库，就进入了资源库了。

3、用终端直接打开Tomcat
(1).进入Tomcat的bin目录下：

(2).授权bin目录下的所有操作：
输入sudo chmod 755 *.sh，回车
(3).输入密码，回车
(4).开启Tomcat，输入sudo sh ./startup.sh，回车

4、到浏览器输入网址localhost，若出现了下面的画面就证明成功了

如何在Markdown中引用本地图片
