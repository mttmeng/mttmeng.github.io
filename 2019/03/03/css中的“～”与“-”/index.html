css中的”～”与”>”

css中”~”

为所有相同的 {父元素中位于 p 元素之后的所有 a 元素}设置背景：

p~a 选择的是前面有 p 元素的每个 a元素。

定义和用法

p~a选择器 p之后出现的所有a。
两种元素必须拥有相同的父元素，但是 a不必直接紧随 p。

css中”>”

css3特有的选择器，A>B 表示选择A元素的所有子元素B。

与A B的区别在于，A B选择所有后代元素，而A>B只选择一代。
.a，.b｛给a，b设置相同的css样式｝；
.a .b｛给a b的所有后代元素设置样式｝；
.a>.b｛给a元素下的子代元素b（只选择一代）设置样式｝;
