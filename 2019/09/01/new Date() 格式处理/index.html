new Date() 格式处理

在一次开发中，我要实现按创建时间倒叙展示的table。但后端在处理数据时将创建时间转化成了字符串，所以我需要将其转化为时间格式来进行比较。

1、将字符串形式的日期转换成日期对象

const strDate=”2019-09-01”; //字符串日期格式
const date= new Date(Date.parse(strDate.replace(/-/g, “/“))); //转换成Data();

2、将时间格式字符串形式的日期转换成时间戳

const strDate=”2019-09-01”; //字符串日期格式
const parseDate= new Date(strDate.replace(/-/g, “/“));
const date= parseDate.getTimer();
