Map和forEach

Map和forEach都属于数组Array的方法，用于遍历数组，高级浏览器支持。

区别

Map有返回值，会返回数组的每一项，但不改变原数组，相当于返回原数组的一个修改过的副本；
而 forEach 没有返回值，只是遍历数组。
从结构和参数形式上看，map方法和forEach方法一模一样，但：

forEach方法

调用数组的每一个元素，并将元素传递给回调函数

currentValue为数组当前item的一个副本，因此改变currentValue不会对原数组产生影响
通过currentArray与index改变原数组

map方法

返回一个新数组，数组中每一个元素为原数组元素调用回调函数处理后的结果。

不会对原数组产生改变
不会对空数组进行检测

findIndex()

findIndex()

findIndex() 方法返回传入一个测试条件（函数）符合条件的数组第一个元素位置。

定义

findIndex() 方法为数组中的每个元素都调用一次函数执行：

当数组中的元素在测试条件时返回 true 时, findIndex() 返回符合条件的元素的索引位置，之后的值不会再调用执行函数。

如果没有符合条件的元素返回 -1

另外需要注意的是：

findIndex() 对于空数组，函数是不会执行的。
findIndex() 并没有改变数组的原始值。

语法：

array.findIndex(function(currentValue, index, arr), thisValue)

currentValue 当前元素，必需
index 可选
。当前元素的索引arr 可选。
当前元素所属的数组对象thisValue 可选。
传递给函数的值一般用 “this” 值。如果这个参数为空， “undefined” 会传递给 “this” 值

返回值：

返回符合测试条件的第一个数组元素索引，如果没有符合条件的则返回 -1。

get, post, put, delete, patch

post = 新增

用于提交请求，可以更新或者创建资源，是非幂等的。多次执行，将导致多条相同的用户被创建（users/1，users/2 …而这些用户除了自增长id外有着相同的数据，除非你的系统实现了额外的数据唯一性检查）。

get = 获取／查看
put = 更新或新增

用于向指定的URI传送更新资源，是幂等的。

put方法用来创建一个URI已知的资源，或对已知资源进行完全替换。

put方法一般会用来更新一个已知资源，除非在创建前，你完全知道自己要创建的对象的URI。

put 会在地址栏显示参数信息，不安全

delete = 刪除
patch = 部分更新

patch 方法是新引入的，是对 put 方法的补充，用来对已知资源进行局部更新。

axios 中常见的 get/delete 请求，也称作query请求。

.push、.unshift、.concat、.Object.Assign

.push、.unshift、.concat、Object.assign

1、push()

向数组的末尾添加一个或多个元素，并返回新的长度。

a、语法:

arrayObject.push(newelement1,newelement2,....,newelementX)
可添加多个元素
eg: const newData = data.push(event);

b、返回值:

把指定的值添加到数组后的新长度。

另:push() 方法把它的参数顺序添加到 arrayObject 的尾部。它直接修改 arrayObject，而不是创建一个新的数组。
push() 方法和 pop() 方法使用数组提供的先进后出栈的功能。

2、unshift()

在数组的开头添加一个或多个元素，并返回新的长度。

a、语法:

arrayObject.unshift(newelement1,newelement2,....,newelementX)
可添加多个元素

b、返回值:

arrayObject 的新长度。

另:unshift() 方法不创建新的创建，而是直接修改原有的数组。

3、concat()用于连接两个或多个数组。

该方法不会改变现有的数组，而仅仅会返回被连接数组的一个副本。

a、语法:

arrayObject.concat(arrayX,arrayX,......,arrayX);

arrayX 必需。该参数可以是具体的值，也可以是数组对象。可以是任意多个。

b、返回值:

返回一个新的数组。

该数组是通过把所有 arrayX 参数添加到 arrayObject 中生成的。

如果要进行 concat() 操作的参数是数组，那么添加的是数组中的元素，而不是数组。

4、Object.assign()
a、作用

1、合并多个对象

2、克隆对象

3、更改属性的值
